# galmoji

[![Build Status](https://travis-ci.org/tamanugi/galmoji.svg?branch=master)](https://travis-ci.org/tamanugi/galmoji)

awesome [GALMOJI](https://en.wikipedia.org/wiki/Gyaru-moji) converter 🎉

# Install

```console
$ npm install galmoji
```

# Usage


```js
var galmoji = require('galmoji');

galmoji('チョーキモチイイ！')
// 于ｮ─≠ﾓ于ｨｨ！
```
